# snail-init

small tools to make creating snail plugins easier!

## usage

just run it to start a new project :p
```bash
bunx snail-init 
```

## args 

- `--plugin-name <name>`: specify the plugin name 
- `--description <description>`: specify the plugin description
- `--icon <icon-url>`: specify the plugin icon url
